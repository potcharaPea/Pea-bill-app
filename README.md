# คิดค่าใบบริการแก้ไฟ - PWA

แอปคิดค่าใบบริการแก้ไฟสำหรับช่างแก้ไฟ ใช้ราคามาตรฐานวัสดุ ปี 2569 ครั้งที่ 3
ทำงานได้ทั้งบนมือถือและคอมพิวเตอร์ ติดตั้งเป็นแอปได้ (PWA) ใช้งานออฟไลน์ได้

## ไฟล์ในโปรเจค

```
PEA-BillApp/
├── index.html              ← หน้าจอหลัก
├── app.js                  ← logic ทั้งหมด (ค้นหา, ตะกร้า, ประวัติ, sync)
├── manifest.json           ← ตั้งค่า PWA (ชื่อแอป, สี, ไอคอน)
├── sw.js                   ← service worker (ใช้งานออฟไลน์)
├── items.json              ← ฐานข้อมูลอุปกรณ์ 727 รายการ จากไฟล์ Excel
├── icons/                  ← ไอคอนแอป (PNG หลายขนาด + SVG)
├── GoogleAppsScript.gs     ← โค้ด backend สำหรับ Google Sheets
└── README.md               ← ไฟล์นี้
```

## วิธีใช้งานแอป

### 1) ทดสอบบนเครื่องตัวเอง (ก่อน deploy)
เปิด Terminal/Command Prompt ในโฟลเดอร์ `PEA-BillApp` แล้วรัน:

```bash
# Python (มีติดมากับเครื่องส่วนใหญ่)
python -m http.server 8000

# หรือ Node.js
npx serve .
```

แล้วเปิดเบราว์เซอร์ไปที่ `http://localhost:8000`

> หมายเหตุ: PWA ต้องใช้ HTTPS จริง ๆ ถึงจะติดตั้งได้ — บน `localhost` ทำได้เลย แต่ถ้าจะ deploy บนเซิร์ฟเวอร์ต้องใช้ HTTPS

### 2) Deploy โฮสต์ฟรี

ทางเลือกง่ายที่สุดสำหรับช่าง/ผู้ใช้ทั่วไป:

**A. GitHub Pages (ฟรี ตลอดชีพ)**
1. สร้าง repository ใหม่บน GitHub (เช่น `pea-bill-app`)
2. อัพโหลดทุกไฟล์ในโฟลเดอร์ `PEA-BillApp` เข้าไป
3. Settings → Pages → Source: `main` branch / `(root)` → Save
4. รออีก 1-2 นาที จะได้ URL เช่น `https://username.github.io/pea-bill-app/`

**B. Netlify Drop (ลากแล้ววาง)**
1. เปิด https://app.netlify.com/drop
2. ลากโฟลเดอร์ `PEA-BillApp` ทั้งโฟลเดอร์เข้าไปในเว็บ
3. ได้ URL ทันที

**C. Cloudflare Pages**
1. https://pages.cloudflare.com/ → Create project → Direct Upload
2. ลากโฟลเดอร์เข้าไป → Deploy

### 3) ติดตั้งลงมือถือ/คอมพิวเตอร์

**Android (Chrome):**
1. เปิด URL ของแอป
2. กดเมนู ⋮ ขวาบน → "Add to Home screen" หรือ "Install app"
3. แอปจะปรากฏบนหน้าจอเครื่องเหมือนแอปทั่วไป

**iPhone/iPad (Safari):**
1. เปิด URL ของแอปใน Safari
2. กดปุ่ม Share (📤)
3. เลื่อนลงเลือก "Add to Home Screen"

**Windows/Mac (Chrome/Edge):**
1. เปิด URL → กดไอคอน "Install" บนช่อง URL (รูปจอ + ลูกศรลง)
2. หรือเมนู ⋮ → "Install คิดค่าใบบริการแก้ไฟ"

## วิธีตั้งค่า Google Apps Script (Optional - สำหรับ sync ข้อมูล)

หากต้องการให้ใบบริการ sync ขึ้น Google Sheets อัตโนมัติ:

1. เปิด https://script.google.com/ → กด "+ New project"
2. ลบโค้ดตัวอย่างทิ้ง วางโค้ดทั้งหมดจากไฟล์ `GoogleAppsScript.gs` ลงไป
3. แก้ค่า `CONFIG.API_KEY` เป็นรหัสลับของคุณเอง (ตัวอักษร+ตัวเลขสุ่ม ๆ)
   ```js
   API_KEY: 'rtAB7xK9pQ2mZ4nF8wV3hL'  // สุ่มเอง 20+ ตัวอักษร
   ```
4. Save (Ctrl+S) → ตั้งชื่อโปรเจค "PEA Bill API"
5. กดปุ่ม "Run" ที่ฟังก์ชัน `setup` (ครั้งแรกจะให้ login + อนุญาตสิทธิ์)
6. ดู Log จะเห็น Spreadsheet URL → จด/บุ๊คมาร์คไว้เพื่อเปิดดูข้อมูล
7. กด **Deploy** → **New deployment**
   - Type (เฟือง): **Web app**
   - Description: PEA Bill API v1
   - Execute as: **Me**
   - Who has access: **Anyone**
8. Authorize → จะได้ URL จบด้วย `/exec` → คัดลอก
9. เปิดแอป PWA → กดเฟือง ⚙ มุมขวาบน → วาง URL และ API Key → กดทดสอบ
10. ถ้าขึ้น "เชื่อมต่อสำเร็จ" คือใช้งานได้!

## ฟีเจอร์ในแอป

- 🔍 **ค้นหาอุปกรณ์เร็ว** – ค้นจากชื่อหรือรหัสพัสดุ พร้อม filter ตามหมวด/ใช้บ่อย
- 🧾 **สร้างใบบริการ** – เพิ่มอุปกรณ์ลงตะกร้า ปรับจำนวน คำนวณราคารวมอัตโนมัติ
- 📜 **ประวัติใบบริการ** – ดูใบบริการที่บันทึกย้อนหลัง พร้อมแชร์/copy ไปต่อได้
- ☁ **Sync Google Sheets** – ใบบริการที่บันทึกจะส่งขึ้น Google Sheets ผ่าน Apps Script
- 🔄 **อัปเดตราคาจาก Google Sheets** – แก้ราคาใน Sheet ครั้งเดียว แอปทุกเครื่องอัปเดตอัตโนมัติ
- 📲 **ใช้งานออฟไลน์** – ติดตั้งเป็นแอปแล้วใช้งานได้แม้ไม่มีเน็ต
- ⚙ **ปรับตัวคูณราคา** – ตั้งค่าตัวคูณราคาผู้ใช้ไฟ (ค่าเริ่มต้น 1.495 = +30% × +15%)

## ใช้ Google Sheets เป็นฐานข้อมูลราคา (อัปเดตได้ตลอด)

หลังตั้ง Google Apps Script + เชื่อมแอปแล้ว สามารถใช้ Sheet เป็นฐานข้อมูลราคาแทน items.json ได้ — แก้ราคาที่ Sheet ครั้งเดียว แอปทุกเครื่องจะดึงราคาใหม่ได้ทันที

### ขั้นตอนนำเข้าข้อมูลครั้งแรก

1. เปิด **Apps Script** (script.google.com) → โปรเจค PEA Bill API
2. ตรวจสอบฟังก์ชัน `importItemsFromUrl` — แก้ค่า `ITEMS_URL` ให้ตรงกับเว็บที่ deploy:
   ```js
   const ITEMS_URL = 'https://potcharapea.github.io/Pea-bill-app/items.json';
   ```
3. ที่แถบบนเลือกฟังก์ชัน **`importItemsFromUrl`** → กด **▶ เรียกใช้** (Run)
4. ดู Log จะขึ้น `นำเข้าสำเร็จ 727 รายการ`
5. **Re-deploy เวอร์ชันใหม่** (Deploy → Manage → ✏ → New version → Deploy)

### เปลี่ยนแอปให้ดึงจาก Sheet

1. เปิดแอป → ⚙ ตั้งค่า
2. เลื่อนลงไปที่ **"แหล่งข้อมูลราคาอุปกรณ์"**
3. เลือก **☁ Google Sheets**
4. แอปจะดึงราคาใหม่จาก Sheet ทันที (cache ไว้ใช้ออฟไลน์ได้ด้วย)

### วิธีอัปเดตราคา / เพิ่มอุปกรณ์

1. เปิด **Google Sheet** "ใบบริการแก้ไฟ - Database" → แท็บ **Items**
2. แก้ราคาในคอลัมน์ `std_price`, `user_price`, `pre_tax` ได้เลย
3. เพิ่มแถวใหม่ที่บรรทัดล่าง → ใส่ code, name, unit, category, ราคา, frequent (ใส่ `Y` ถ้าใช้บ่อย)
4. ลบแถว = ลบทั้งแถว (คลิกขวา → Delete row)
5. ในแอป → ⚙ → กด **🔄 โหลดราคาใหม่** → ราคาอัปเดตทันที

### โครงสร้างคอลัมน์ใน Sheet "Items"

| คอลัมน์ | ความหมาย | ตัวอย่าง |
|---------|----------|---------|
| code | รหัสพัสดุ (ห้ามซ้ำ) | 1020020002 |
| name | ชื่ออุปกรณ์ | CONDUCTOR,ACSR 50/8 sq.mm. |
| unit | หน่วย | ม. / ชิ้น / ม้วน |
| category | หมวด (ใช้กรองในแอป) | หมวด 1 (สาย/อุปกรณ์ไฟฟ้า) |
| std_price | ราคามาตรฐาน | 22.50 |
| user_price | ราคาคิดผู้ใช้ไฟ | 33.6375 |
| pre_tax | ราคาก่อนภาษี | 29.25 |
| frequent | `Y` = ขึ้นในแท็บ "ใช้บ่อย" |  Y |

### ข้อดีของการใช้ Sheet เป็นฐานข้อมูล

- ✅ แก้ราคา/เพิ่มอุปกรณ์ได้จากที่ไหนก็ได้ ไม่ต้อง re-upload โค้ด
- ✅ ช่างหลายคนใช้แอปเดียวกัน → อัปเดตราคาที่ Sheet ครั้งเดียว ทุกคนเห็นพร้อมกัน
- ✅ มีประวัติการแก้ไขใน Google Sheet (View → Version history)
- ✅ แชร์ Sheet ให้คนอื่นดู/แก้ได้ตามสิทธิ์ที่กำหนด

### ข้อควรระวัง

- ⚠ ลบแถวข้อมูล แต่ห้ามลบ **บรรทัดที่ 1 (header)** เด็ดขาด
- ⚠ คอลัมน์ต้องเรียงตามตารางข้างบน (อย่าสลับลำดับ)
- ⚠ หลังแก้ Sheet กดปุ่ม **🔄 โหลดราคาใหม่** ในแอปก่อน ราคาใหม่จึงจะมา

## โครงสร้างราคา

จากการตรวจสอบไฟล์ Excel:
- **ราคามาตรฐาน** (std_price) = ราคาทุนตามประกาศ ปี 2569 ครั้งที่ 3
- **ราคาก่อนรวมภาษี** (pre_tax) = std_price × 1.30 (บวกค่าดำเนินการ 30%)
- **ราคาคิดผู้ใช้ไฟ** (user_price) = pre_tax × 1.15 (บวก factor อีก 15%)
- รวมตัวคูณ = **1.495** (สามารถปรับได้ในหน้า "ตั้งค่า")

**หมายเหตุ:** หมวด 1 (สาย/อุปกรณ์ไฟฟ้า) ใช้ค่าราคา "คิดจริงกับผู้ใช้ไฟ" ที่ระบุในไฟล์ Excel เป็นหลัก
ส่วนหมวดอื่น (Optic Fiber, Duct Bank, คอนกรีต) ใช้ตัวคูณข้างต้น

## Tips

- กดที่ใบบริการในประวัติเพื่อดูรายละเอียด/แชร์/ลบ
- กดปุ่ม "Sync" ในแท็บประวัติ เพื่อ push ใบบริการที่ยังไม่ sync ทั้งหมด
- หากราคามีการเปลี่ยนแปลง: อัปเดตไฟล์ `items.json` แล้วกด "โหลดราคาใหม่" ในหน้าตั้งค่า

## ปัญหาที่พบบ่อย

| ปัญหา | วิธีแก้ |
|------|--------|
| แอปติดตั้งไม่ได้ | ต้อง deploy ผ่าน HTTPS (GitHub Pages / Netlify ใช้ HTTPS โดยอัตโนมัติ) |
| Sync ไม่ผ่าน | ตรวจ URL ลงท้ายด้วย `/exec` และ API Key ตรงกับใน Apps Script |
| ติดตั้งแล้วราคาไม่อัปเดต | กด "ตั้งค่า" → "โหลดราคาใหม่" หรือ uninstall + reinstall แอป |
| iOS ไม่เห็นปุ่ม Install | iOS ใช้ Safari → Share → Add to Home Screen เท่านั้น |
